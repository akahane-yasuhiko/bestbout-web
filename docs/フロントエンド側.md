# 🥊 BestBout Frontend Summary

## ① プロジェクト全体構成（目的・概要）

**BestBout** は、格闘技イベント（例：RIZIN、UFCなど）の各カードに対して
ファンの試合予測コメントや的中率を集計・表示するWebアプリです。

### 🎯 目的

* 各試合カードの勝敗予想・分析コメントを集約
* どのユーザーの予想が的中したかを可視化
* ファン同士の議論や傾向分析を促進する

### 🧩 全体アーキテクチャ

| 層                      | 技術                           | 内容                                                 |
| ---------------------- | ---------------------------- | -------------------------------------------------- |
| **Frontend (本ドキュメント)** | React + Vite + TypeScript    | SPA構成。GitHub Pages で静的配信                           |
| **Backend**            | Cloud Run (Python/FastAPI予定) | 予測解析・データ収集API（別途まとめ済）                              |
| **Data Source**        | JSON / Cloud Storage / API   | 試合カード・ユーザー投稿データ                                    |
| **Hosting**            | GitHub Pages (Project Pages) | `https://akahane-yasuhiko.github.io/bestbout-web/` |

### 🧱 ページ構成（想定）

| ページ         | 内容                               |
| ----------- | -------------------------------- |
| `/`         | トップページ。ポスター風カルーセル表示（RIZIN風背景）    |
| `/cards`    | 試合カードリストとカルーセル。各カードの概要と日程        |
| `/card/:id` | 個別試合ページ（予測コメント一覧をTogetter風に表示予定） |
| `/rankings` | 的中率ランキング（今後実装予定）                 |

## ② フロントエンド実装内容（ここまで）

### 🔧 開発環境

* **フレームワーク**：React + TypeScript + Vite
* **デプロイ**：GitHub Actions → GitHub Pages (Project Pagesモード)
* **ルーティング**：react-router-dom
* **ビルド設定**：

  * `vite.config.ts` に `base` を `import.meta.env.BASE_PATH` から取得
  * Actions内で `BASE_PATH=/bestbout-web/` を `env` 経由で渡してbuild

---

### 🎨 UI実装一覧

#### 1️⃣ **トップページ（Home）**

* `PostersCarousel.tsx` + `CardPoster.tsx`
* **RIZIN風赤×黒ポスター背景（共通WebP画像）** をベースに
  各試合カードタイトル・日付を上から重ねて表示
* **Keen Slider** により3〜4枚を横並び自動スライド
* `2:3` 縦長比（600×900 / 1000×1500）を採用

```tsx
const posters = [
  { id: 'p1', title: 'Main Event — 桜庭隼人 vs リカルド・フローレス', date: '2025-11-08' },
  { id: 'p2', title: 'Co-main — 田村龍也 vs レオン・バスケス', date: '2025-11-08' },
  { id: 'p3', title: 'アレックス・コールマン vs 岩城剛', date: '2025-11-22' },
  { id: 'p4', title: 'ユーリ・サヴィチェンコ vs 近藤海翔', date: '2025-12-06' },
]
```

---

#### 2️⃣ **カード一覧ページ（Cards）**

* `CarouselCards.tsx` + `CardPoster.tsx`
* `cards.sample.json` から試合カードデータをロード
* カルーセル表示：スマホ1枚、PC3〜4枚表示
* 各カードのタイトル・日付を共通背景上にオーバーレイ
* API未接続時はローカルサンプルを使用

```ts
// api.ts 抜粋
const r = await fetch(`${import.meta.env.BASE_URL}data/cards.sample.json`);
return r.json();
```

---

#### 3️⃣ **汎用ポスターコンポーネント**

* `CardPoster.tsx`：全カードで共通の背景を使い、title/dateを上から重ねる
* 背景画像：`public/images/posters/generic-fight-poster.webp`（2:3比）
* 下部に暗いグラデーションを重ねて文字を視認性よく表示

```tsx
const src = imageSrc ?? `${import.meta.env.BASE_URL}images/posters/generic-fight-poster.webp`
```

---

#### 4️⃣ **スタイル関連**

* `keen-slider` + カスタムCSS
* ポスター風スタイル：赤×黒グラデ＋対峙シルエット（RIZIN風）
* ダークテーマ基調、柔らかいドロップシャドウ・グラデ重視

---

#### 5️⃣ **サンプルデータ**

* `cards.sample.json`：試合カード一覧
* `predictions.sample.json`：ユーザーコメント（Togetter風UI用）
